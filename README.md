# focus-study-figma

## Descrição
Focus Study é um aplicativo mobile prototipado no Figma para ajudar estudantes a organizar seus estudos e manter o foco. Reúne temporizador Pomodoro, gerenciamento de tarefas e metas diárias em uma interface simples e intuitiva.
